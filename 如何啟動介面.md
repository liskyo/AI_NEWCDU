# 如何開啟 Web UI 介面 (Windows)

本專案主要設計於 Linux 環境運行，但在 Windows 環境下亦可透過以下步驟啟動進行開發或測試。

## 快速啟動 (推薦)

我已為您建立了 **`windows_run.bat`** 腳本，位於專案根目錄。

1.  雙擊執行 **`windows_run.bat`**。
2.  腳本會自動檢查環境：
    *   若無虛擬環境，會自動建立 `windows_venv` 並安裝所需套件 (`requirements.txt`)。
3.  自動開啟兩個視窗分別執行：
    *   **RestAPI** (Port 5001)
    *   **WebUI** (Port 5501)
4.  啟動完成後，請打開瀏覽器訪問：
    *   [http://localhost:5501](http://localhost:5501)

## 手動啟動步驟

若您希望手動執行，請依序操作：

1.  **建立並啟用虛擬環境**:
    ```cmd
    python -m venv windows_venv
    windows_venv\Scripts\activate
    ```

2.  **安裝依賴套件**:
    ```cmd
    pip install -r RestAPI\requirements.txt
    pip install -r webUI\requirements.txt
    ```

3.  **啟動 RestAPI (後端)**:
    *   開啟新的 CMD 視窗
    *   `cd RestAPI`
    *   `..\windows_venv\Scripts\python app.py`
    *   確認看到 `Running on http://0.0.0.0:5001`

4.  **啟動 WebUI (前端介面)**:
    *   開啟新的 CMD 視窗
    *   `cd webUI`
    *   `..\windows_venv\Scripts\python web\app.py`
    *   確認看到 `Running on http://0.0.0.0:5501`

## 常見問題

*   **缺少模組**: 若出現 `ModuleNotFoundError`，請確認是否已啟用虛擬環境並執行 `pip install`。
*   **Port 被佔用**: 若 5001 或 5501 無法啟動，請檢查是否有其他程式佔用該連接埠。
